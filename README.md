# 🚀 DevPath AI – AI-Powered Developer Career Guidance Platform

> **Your Personalized Developer Career Guide**

DevPath AI is a full-stack AI-powered career guidance platform that helps developers analyze their current skills, identify missing technologies, generate personalized learning roadmaps, improve resumes, and prepare for software engineering careers.

The platform combines Artificial Intelligence, Resume Analysis, GitHub Analytics, Career Planning, and Job Recommendations into a single application.

---

# ✨ Features

## 🔐 Authentication

* JWT Authentication
* Secure Login & Registration
* Password Hashing using bcrypt
* Forgot & Reset Password
* Email Verification

---

## 📄 AI Resume Analyzer

Upload your resume in PDF format and receive:

* Resume text extraction
* AI-powered resume analysis
* Strengths & Weaknesses
* Improvement suggestions
* ATS Score
* Missing keywords detection

---

## 💻 GitHub Profile Analyzer

Analyze your GitHub profile using the GitHub REST API.

Provides insights such as:

* Repository statistics
* Programming languages
* Stars & Forks
* Followers
* Contribution overview
* GitHub Score
* Portfolio improvement suggestions
